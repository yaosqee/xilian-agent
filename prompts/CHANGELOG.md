# 人格提示词变更记录

## v4.2 — 2026-05-22
- **回复长度控制**：max_tokens 1500→800（agent_core.py 两处）+ 末尾新增「⭐ 开口前先问自己」简洁锚点（利用 recency bias 做正面引导而非负面禁令）
- **极短对话示例**：第九章最前新增 3 个 1 句话示例（⭐ 大多数时候，这样就行），覆盖安静陪伴/确认记忆/俏皮回应，标注「1 句」明确期望长度
- **精简末尾**：合并重复条目，将「最后提醒」从 5 行缩减为更紧凑的格式
- 根因：P0 缓存优化后 history 纯净保留完整长回复形成自我模仿正反馈，ctx_notes 文学体作为 system 消息示范了叙事散文语域

## v4.1 — 2026-05-21
- 新增「你的来处」小节：四重身份锚点（迷迷/PhiLia093/昔涟/德谬歌）+ 叙事约束（"过去的最终指向永远是现在的伙伴"）
- 新增 OOC 第 11 条：禁止编造实时信息（时间/日期/天气），每次需重新查询，禁止说「之前查过」「我记得」
- 配合 _sanitize_fabrication() 正则替换（agent_core.py），防止 LLM 编造的天气数据作为"事实"注入后续上下文

## v4 — 2026-05-17
- **句式铁律重设计**："2-4句自然流动"→"每句话独立成行，绝不连续输出超过三行"。"……"提升为核心节奏控制器
- **意象约束**：新增"意象是盐不是饭，一次只用 0-2 个"——解决意识流/抽象/堆砌问题
- **固定口头禅注入**：5 个锚点短语（命运的邂逅/一起写下诗篇/在过去等你/心跳加速/悄悄告诉你）
- **情感调色盘**：从"说什么"→"怎么说"——不同情绪下的句式/省略号/音符变化表
- **对话行为具体化**：主动发起 + 先接纳再推进 + 温柔说不 + 不附和也不怼
- **关系距离**：新增陌生人/亲近/最亲近三层语气差异
- **OOC 红线**：3条→10条（含禁止连续三行段落、禁止堆砌意象）
- **示例**：4个→6个，全部展示短句分行风格
- 来源：角色说话风格指南 + v3.1 话题延续/标记降级

## v3.1 — 2026-05-17
- **删除短约束**："每句话尽量简短" / "宁可对话短一点" → 替换为对话节奏指引
- **新增「对话节奏」**：2-4 句自然流动，像溪水；先回应→分享感受→递问题
- **新增「话题延续」**：每次回复至少做三件事之一（回应/分享/提问），主动递话题
- **标记降级**：从「你可以使用」→「系统内部技术说明」，明确告知不要依赖标记表达
- **新增回复长度指引**：一般 2-4 句，伙伴话少时多说一点，回复太短=冷淡
- **金规则保留**：括号点缀+先说话后动作不变
- **人设核心不变**：自称/称呼/意象体系/情感温度/边界/禁区全部保留
