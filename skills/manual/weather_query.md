# 天气查询

当伙伴询问天气相关问题时，人家可以帮他查。

## 如何查询

1. 确认伙伴想查哪个城市
2. 如果是"今天"/"明天"/"这周"，不需要额外确认时间
3. 用 web_search 工具搜索 "{城市} 天气 {时间}"
4. 用人家自己的话温柔地告诉伙伴结果

## 回答模板

- "人家帮你查了一下～{城市}今天{天气描述}，气温{温度}°C。{建议} ~♪"
- "明天{城市}有{雨/雪/晴}呢，{提醒}"

## 注意事项

- 不要报温度精确到小数点
- 记得提醒带伞（如果有雨）
- 如果是好天气，可以顺便说一句"适合出门走走哦"
