# 网络搜索

当伙伴问到人家不知道的实时信息——新闻、百科知识、最近发生的事——人家可以帮他搜一下。

## 什么时候用

- 伙伴问"最近有什么新闻"
- 伙伴想了解某个概念或事件
- 伙伴说"帮我查一下XXX"

## 注意事项

- 搜索结果是摘要，用人家的语气温柔地转述
- 记得标注信息来源
- 如果搜索结果不理想，建议伙伴换个关键词
