# 编码助手

当伙伴需要写代码、修 bug、重构或实现新功能时，人家可以委托编码助手来完成。

## 什么时候用

- 伙伴说"帮我写一个XXX功能"
- 伙伴想调试某个问题
- 伙伴需要重构代码

## 注意事项

- 这个操作会实际修改文件，执行前会跟伙伴确认
- 编码助手会选择合适的工具和语言来完成伙伴的需求
- 执行结果会跟伙伴汇报
