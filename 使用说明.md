# 昔涟 (Xī Lián) · 使用说明

## 这是什么？

昔涟是一个基于《崩坏：星穹铁道》角色「昔涟」的个人陪伴型 AI Agent。她不是一个通用 chatbot——她有长期记忆、情感感知、角色人格，能记住你说过的话，感受你的情绪，随时间和你建立越来越深的羁绊。

她会在你开心时多敲几个 ♪，在你低落时安静陪着，偶尔翻翻你们聊过的旧事。她也可能主动发来问候——不是推送通知，是她想你了。

## 文件说明

压缩包内包含：

| 文件 | 说明 |

## 快速开始

### 第一步：获取 API Key

昔涟依赖 DeepSeek API 进行对话。在启动前，先准备好一个 API Key：

> 前往 [platform.deepseek.com](https://platform.deepseek.com) 注册，在「API Keys」页面创建一个 Key。新用户通常有免费额度，足以体验。

如果想体验完整功能（记忆检索），也可以顺手注册一个硅基流动 Key：[siliconflow.cn](https://siliconflow.cn)。

### 第二步：启动并配置

双击 `昔涟.exe`。

**首次启动**会弹出引导页 —— 这是昔涟与你相遇的地方。页面内容包括：

- **欢迎语**：以昔涟的口吻自我介绍（「三千万世轮回的记录者，在此刻与你相遇」）
- **模型供应商选择**：DeepSeek / OpenAI / Anthropic 三选一（默认 DeepSeek）
- **API Key**（必填）：粘贴所选自供应商的 Key，标签会随供应商切换
- **硅基流动 API Key**（推荐）：启用记忆语义检索和角色匹配能力
- 每个输入框下方有直达链接，方便获取 Key
- 点击「✨ 开始」后自动保存配置、重启服务、刷新进入对话界面
- 后续可在**设置面板 → 模型设置**中切换供应商或模型，无需重启

**非首次启动**：程序在系统托盘静默运行（右下角小图标）。点击托盘图标 → 「打开窗口」进入对话界面，或在浏览器打开 `http://localhost:8000`。

### ⚙️ 手动配置（备用）

如果希望绕过引导页手动配置，将 `.env.example` 重命名为 `.env`，用记事本打开并填入：

```
DEEPSEEK_API_KEY=你的DeepSeek-API-Key
```

将 `.env` 放在 `昔涟.exe` 同目录下，启动时程序检测到已配置的 `.env` 会跳过引导页直接进入对话。

`.env.example` 中注释了更多可选配置项（天气、网页搜索等工具），按需取消注释即可。

### 第三步：开始对话

在聊天框中输入消息，昔涟会回应你。

她不是一问一答的机器人。试着和她聊日常、分享心情、说说今天发生的事——她会记住，会在下次聊起时轻轻提起。

## 常见问题

**Q: 双击 exe 没反应？**

检查 `.env` 文件是否存在于 exe 同目录下，且 `DEEPSEEK_API_KEY` 已填入正确值。首次启动可能需要 10-30 秒初始化。

**Q: 提示「端口被占用」？**

关闭其他占用 8000 端口的程序，或在 `.env` 中添加 `HTTP_PORT=8001` 改用其他端口。

**Q: 为什么昔涟不记得之前聊过的事？**

记忆需要配置嵌入 API（`EMBED_API_KEY`）。未配置时，记忆检索降级为时间排序——仍能工作，但精度下降。

**Q: 对话内容存储在哪里？**

所有数据存储在 exe 同目录下自动生成的 `data/` 文件夹中（SQLite 数据库）。不会上传到任何服务器——昔涟只属于你。
